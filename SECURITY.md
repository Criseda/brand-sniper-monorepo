# Security Policy

## Supported Versions

Only the latest release on the `main` branch of this repository receives security updates.

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |
| < Latest| No        |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do not report it publicly via a GitHub issue.

Please report it securely through GitHub's built-in **Private Vulnerability Reporting** feature:

1. Navigate to the repository on GitHub.
2. Click on the **Security** tab.
3. Click on **Advisories** under the "Vulnerability reporting" section on the left sidebar.
4. Click the **Report a vulnerability** button to open a draft security advisory privately to the maintainers.
